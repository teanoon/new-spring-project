# new-spring-project
This is a java boilerplate project using spring and gradle with eclipse-support.

# Features
* Basic MVC structure with a service layer
* Test-framework: jUnit
* Data Persistence:
  * Hibernate integration
  * Migration strategy
  * Annotation configuration

# Working
* Enhanced MVC:
  * CRUD
  * Responders
  * Exceptions
